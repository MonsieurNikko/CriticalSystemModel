# Demo de faisabilite

Ce dossier contient une visualisation HTML/JS autonome du coeur du projet :

- deux trains concurrents ;
- un troncon critique partage ;
- les places du reseau de Petri ;
- les transitions et messages associes ;
- les invariants affiches en direct.

Ouvrir `demo/index.html` dans un navigateur suffit. Aucune installation n'est necessaire.

Cette demo ne remplace pas les tests Scala ni l'analyseur Petri : elle sert a rendre le protocole lisible pendant une presentation.
